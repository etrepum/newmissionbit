Intro Computer Programming

In the 'Intro to Computer Programming' class, students will learn HTML5, CSS3 and Javascript and some core programming concepts through project based instruction. Within the semester the class will build websites and interactive applications. At the end of the semester, the class is broken up into teams, each paired with an instructor and spends a few weeks building a final project. Projects in the Fall semester included an HTML5 game, an interactive light show that change the hue of lights based on sound frequency, a browser-based drawing tool and an instant message app.

Intermediate Computer Programming

The core work in this class will be using Ruby and Sinatra as well as some SQL and Javascript to build more complex web apps than the intro class. This class will dive deeper into core programming concepts through project based instruction.
